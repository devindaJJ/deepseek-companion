# 🤖 DeepSeek AI Companion

A Python-based AI assistant using DeepSeek's API with a clean, modular architecture.

## ✨ Features

- **Class-based design** with separation of concerns
- **Multiple AI providers** support (DeepSeek, Groq, Gemini, Ollama)
- **Configuration management** with environment variables
- **Conversation history** management
- **Streaming responses** support
- **Logging system** for debugging
- **Command-line interface** with argparse
- **Example usage** included

## 🚀 Quick Start

### Prerequisites
- Python 3.8+
- pip (Python package manager)
- DeepSeek API key (or alternative provider)
